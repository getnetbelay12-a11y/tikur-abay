# Tikur Abay Driver App

Flutter mobile app for the Tikur Abay customer and driver experience.

## Current Focus

- OTP-first login for customers and drivers
- Driver KYC-gated onboarding
- Policy-driven document upload, preview, and focus flows
- Role-based customer and driver shells

## Presentation Guide

For the strongest live demo path, use:

- [Mobile Presentation Walkthrough](../../docs/mobile-presentation-walkthrough.md)

## Development

Run from `apps/driver`:

```bash
flutter pub get
flutter run
```

Verification:

```bash
flutter test
flutter analyze
```
