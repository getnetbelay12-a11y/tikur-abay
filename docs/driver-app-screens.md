# Driver App Screens

- `LoginScreen`: email/phone login and secure token bootstrap
- `AssignedTripScreen`: active trip, checkpoint progress, and background GPS note
- `TripDetailScreen`: shipment details, route, checkpoint logging, issue report
- `StatusUpdateScreen`: loading, loaded, transit, border, offloading, delivered
- `ReportIssueScreen`: structured report flow for fuel, tire, maintenance, accident, obstacle, breakdown, delay, and support requests
- `ChatScreen`: dispatcher chat, trip room, image/file attachments
- `DocumentScreen`: uploaded trip docs, POD upload, vehicle and personal documents
- `NotificationsScreen`: maintenance instructions, dispatcher alerts, and escalated field reports
- `ProfileScreen`: driver identity, license expiry, assigned vehicle, emergency contacts
