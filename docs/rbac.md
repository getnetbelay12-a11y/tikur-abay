# Roles and Permissions

- Super Admin: full system access
- Operations Manager: full operations visibility, dispatch approval, dashboards
- Dispatcher: assignment, dispatch, trip monitoring
- Branch Officer: branch-level intake, documents, and shipment updates
- Driver: assigned trips, GPS, proofs, incidents
- Customer: own shipments, agreements, documents, proofs
- Compliance Officer: compliance review, document approval, agreement audit
- Executive Viewer: read-only dashboards and reports
